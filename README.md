QirkDB
======

Dependencies

- [Go](http://golang.org/)
- [gvp](https://github.com/pote/gvp): Go Versioning Packager (like rvm)
- [gom](https://github.com/mattn/gom): Go package versioning (like bundler)

Installing dependencies

> $ gom install

> $ gom -test install

For watching your files and run all tests.
